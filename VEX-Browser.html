VEX Browser

Simple Secure Private

Creating the VEX internet browser was pretty easy but it also makes the browser snappy and more secure than others because it doesn't actually store any data that you browse: Cookies, Search history etc. It doesn't have much as of yet but soon im sure there will be more to it.

Uses google by default

The url bar doesnt actually search with a search engine, It only uses the full url of a website to enter it, But you can set a default url to start up on when you launch so that you dont need to use google or you could just have no start page and be completly invisible to search engine companies.

Fuctions Buttons

Because of its simplicity it only has 5 buttons on the whole ui: Back, Forward, Refresh, Google Search (takes you to google.com) and YouTube, which is just a hyperlink to YouTube.

Functionality

VEX browser can stream video of any site as long as the website doesn't require a certain browser for its service, apart from that it would work just as good as Google Chrome or Microsoft Edge.

FIRTACH

Fully Intelligent Robot That Always Can Help

FIRTACH's Original idea was to have it to answer basic things with just text input. but now he can answe basic queries and use machine learning to pick out words out of an intents file to chat to you like a chatbot FIRTACH is still being worked on but should soon be like the original idea and be able to answer queries, equations, chat to you and also be able to automate things like lights and other smart home devices.

Version 0.1 - 0.3

In the early days FIRTACH (Version 0.1) Just didn't work, It was only a glitchy file that just didn't work out because it was python in a .batch file. It was also called 'VEX Assistant' because i didn't know what to call it yet.

Version 0.4

In these versions FIRTACH kind of worked and could use basic functions like telling the time and date using user text as the input.

Version 0.6 - 1.0

These versions use Discord's bot api to allow any user in a server with FIRTACH to ask it questions and to get picked out responses using a machine learning response module called neural intents by NeuralNine. These versions also became the first ones that actually worked better than all the previous versions that only broke after one query.

Version 1.2

This version is the first one close to the original idea, It uses the users voice as the input and uses a module called pyttsx3 for the voice output (tts).
